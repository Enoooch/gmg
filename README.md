# Gatsby Markdown Generator
Generate markdown file from cmd

[Gatsby-starter-blog](https://github.com/gatsbyjs/gatsby-starter-blog) template is needed.

## Usage
```
$ git clone https://github.com/Enoooch/gmg.git
```

```
$ npm link
```

Run by calling:
```
$ gmg
```
